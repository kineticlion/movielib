# MovieLib

A react Interface designed for management of movies.

# Packages

- bootstrap v 4.1.1
- font-awesome v 4.7
- lodash v 4.17.10
- prop-types v 15.6.2

# Features

- Load, Delete, Like movies in the Movie Library
- Sorting, Filtering, Pagination of the Movies

# How To Use

- Run 'npm start' to deploy development server
